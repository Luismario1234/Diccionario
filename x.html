Lenguaje utilizado para localizar elementos dentro de documentos XML.
N° palabra Español Ingles
1 XML (eXtensible Markup Language) Lenguaje de marcado utilizado para almacenar e intercambiar datos estructurados. A markup language used to store and exchange structured data.
2 XPath A language used to locate elements within XML documents.
3 XQuery Lenguaje diseñado para consultar y manipular datos XML. A language designed to query and manipulate XML data.
4 XML Schema Definition (XSD) Estándar para definir la estructura y validación de documentos XML. A standard used to define the structure and validation of XML documents.
5 XAMPP Paquete de software que incluye servidor web, base de datos y herramientas para desarrollo. A software package that includes a web server, database, and development tools.
