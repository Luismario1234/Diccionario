N° Palabra Español Ingles
1 Garbage Collector Proceso automático de gestión de memoria que identifica y destruye objetos de software que ya no están en uso. Automatic memory management process that identifies and destroys software objects no longer in use.
2 Geo-replication Práctica de replicar bases de datos en diferentes regiones geográficas para reducir la latencia y asegurar la recuperación ante desastres. Practice of replicating databases across different geographic regions to reduce latency and ensure disaster recovery.
3 gRPC(Remote Procedure Calls) Framework de código abierto de Google para llamadas a procedimientos remotos, utilizado para la comunicación rápida entre microservicios. Google open-source framework for remote procedure calls, used for fast communication between microservices.
4 Graceful Degradation Capacidad de una arquitectura para mantener sus funciones esenciales activas cuando partes del sistema dejan de responder. Ability of an architecture to keep essential functions active when parts of the system stop responding.
5 Governance (Gobierno de Datos) Conjunto de políticas, procesos y estándares que aseguran el uso correcto, seguro y eficiente de los datos de una empresa. Set of policies, processes, and standards ensuring the correct, secure, and efficient use of company data.
