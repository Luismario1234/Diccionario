N° Palabra Español Ingles
1 Java An object-oriented programming language used to develop web, mobile, and desktop applications. Lenguaje de programación orientado a objetos utilizado para desarrollar aplicaciones web, móviles y de escritorio.
2 JavaScript A programming language used to create dynamic and interactive web pages. Lenguaje de programación utilizado para crear páginas web dinámicas e interactivas.
3 JDBC (Java Database Connectivity) A technology that allows Java applications to connect to databases. Tecnología que permite conectar aplicaciones Java con bases de datos.
4 JDK (Java Development Kit) A set of tools required to develop and run Java applications. Conjunto de herramientas necesarias para desarrollar y ejecutar aplicaciones Java.
5 JSON (JavaScript Object Notation) A lightweight format used to store and exchange data between applications. Formato ligero para almacenar e intercambiar datos entre aplicaciones.
