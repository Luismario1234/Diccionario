N° Palabra Español Ingles
1 Heuristics Método de optimización en bases de datos utilizado por el motor de consultas para elegir el camino más rápido para leer datos. Database optimization method used by the query engine to choose the fastest path for reading data.
2 Horizontal Scaling Estrategia de crecimiento que consiste en agregar más servidores económicos a la infraestructura en lugar de mejorar uno solo. : Growth strategy consisting of adding more budget-friendly servers to the infrastructure instead of upgrading a single one.
3 Hadoop Entorno de software de código abierto diseñado para el almacenamiento y procesamiento distribuido de conjuntos de datos masivos (Big Data). Open-source software framework designed for distributed storage and processing of massive datasets (Big Data).
4 Heartbeat Señal periódica enviada entre servidores de un clúster para confirmar mutuamente que siguen encendidos y funcionando bien. Periodic signal sent between servers in a cluster to mutually confirm they remain powered on and functioning properly
5 Hot Spotting Problema de rendimiento en bases de datos distribuidas donde un solo nodo recibe todo el tráfico de lectura o escritura de golpe. Performance issue in distributed databases where a single node suddenly receives all the read or write traffic.
