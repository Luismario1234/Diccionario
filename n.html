N° Palabra Español Ingles
1 Nodo (Node) Computadora, servidor o instancia individual dentro de una red o sistema distribuido que procesa datos. A computer, server, or individual instance within a network or distributed system that processes data.
2 NoSQL Categoría de sistemas de gestión de bases de datos que no utilizan el modelo relacional tradicional (tablas). A category of database management systems that do not use the traditional relational model (tables)
3 Normalización Proceso de organizar las tablas de una base de datos para reducir la redundancia y mejorar la integridad. The process of organizing database tables to reduce redundancy and improve data integrity.
4 N-Capas (N-Tier) Patrón arquitectónico que divide una aplicación en múltiples niveles físicos o lógicos independientes An architectural pattern that divides an application into multiple independent physical or logical levels
5 Null (Nulo) Indicador en bases de datos que representa la ausencia de valor o un dato desconocido en un campo. An indicator in databases that represents the absence of a value or an unknown data field.
