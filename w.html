N° palabra Español Ingles
1 Webhooks: Mecanismo automatizado de comunicación asíncrona que envía datos en tiempo real de una aplicación a otra inmediatamente cuando ocurre un evento específico. : Automated asynchronous communication mechanism that sends real-time data from one application to another immediately when a specific event occurs.
2 WebSockets : Bidirectional, low-latency communication protocol that keeps a persistent open connection between the client and the server for real-time data
3 Workflow (Flujo de Trabajo) Orquestación y secuencia de tareas, reglas de negocio y pasos automatizados que un sistema ejecuta para completar un proceso operativo. Orchestration and sequence of automated tasks, business rules, and steps that a system executes to complete an operational process.
4 WAF (Web Application Firewall): Filtro de seguridad especializado que monitorea y bloquea el tráfico HTTP malicioso dirigido a una aplicación web para prevenir ataques comunes. Specialized security filter that monitors and blocks malicious HTTP traffic directed at a web application to prevent common attacks.
5 WAL (Write-Ahead Logging) Técnica de base de datos donde los cambios se escriben primero de forma segura en un archivo de registro (log) antes de aplicarse a la base de datos principal. Database technique where changes are first written securely to a log file before being applied to the main database.
