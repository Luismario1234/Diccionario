N° Palabra Español Ingles
1 Internet A global network that connects millions of devices to share information and access digital services. Red mundial que conecta millones de dispositivos para compartir información y acceder a servicios digitales.
2 Interface (Interfaz) A means that allows interaction between the user and a computer system or application. Medio que permite la interacción entre el usuario y un sistema o aplicación informática.
3 Input (Entrada) Data or information entered into a system for processing. Dato o información que se introduce en un sistema para ser procesada.
4 IDE (Integrated Development Environment) An Integrated Development Environment that provides tools to write, debug, and run code. Entorno de Desarrollo Integrado que proporciona herramientas para programar, depurar y ejecutar código.
5 Inheritance (Herencia) An object-oriented programming mechanism that allows a class to inherit characteristics and methods from another class. Mecanismo de la programación orientada a objetos que permite que una clase herede características y métodos de otra.
