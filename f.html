1 Feature Flag Técnica que permite activar o desactivar funcionalidades específicas en producción sin necesidad de desplegar código nuevo. Technique that allows enabling or disabling specific features in production without deploying new code.
2 Fault Tolerance Capacidad de un sistema informático para seguir operando correctamente incluso si uno o más de sus componentes fallan. Ability of a computer system to continue operating correctly even if one or more of its components fail.
3 Federated Database Sistema que integra múltiples bases de datos autónomas y geográficamente dispersas para que parezcan una sola base lógica. System that integrates multiple autonomous, geographically dispersed databases to look like a single logical database.
4 Firebase Plataforma de desarrollo de Google que incluye bases de datos NoSQL en tiempo real, autenticación y almacenamiento en la nube. Google development platform that includes real-time NoSQL databases, authentication, and cloud storage.
5 Foreign Key Constraint Regla de integridad en bases de datos que impide eliminar registros si existen dependencias vinculadas en otra tabla. : Integrity rule in databases that prevents deleting records if linked dependencies exist in another table.
